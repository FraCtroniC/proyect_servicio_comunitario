# Código de Conducta

Se espera que todos los participantes en este proyecto mantengan un comportamiento respetuoso y profesional.

Reglas básicas:
- Respeta a las personas y sus ideas.
- No toleramos acoso, discriminación ni lenguaje ofensivo.
- Mantén las discusiones centradas en el trabajo técnico.

Reporte de incidentes: contacta al mantenedor del proyecto (reemplaza con email) si observas comportamientos que violen este código.
